# Baixador de vídeos para Android

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)
[![Kotlin](https://img.shields.io/badge/Kotlin-100%25-purple.svg)](https://kotlinlang.org/)

O **Baixador de Vídeos** é um aplicativo Android baseado no projeto open-source [Seal](https://github.com/junkfood02/Seal) (versão 1.13.1). O Seal é um downloader que utiliza o [yt-dlp](https://github.com/yt-dlp/yt-dlp) para ler plataformas de vídeo.

Este projeto é um *fork* do Seal que mantém grande parte das funcionalidades originais e **adiciona novas funções** não disponíveis na versão 1.13.1.

## 🛠️ Tecnologias Utilizadas

- **Kotlin** – 100% do código
- **Jetpack Compose** – UI moderna e declarativa
- **Material Design 3** – Design system do Google
- **yt-dlp** – Motor de download principal
- **aria2c** – Downloader externo para aceleração
- **mutagen** – Extração e inserção de metadados em áudio
- **Gradle (Kotlin DSL)** – Sistema de build
